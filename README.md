# VetLink

**Smart Pet Health & Vaccination Tracker**

VetLink is a modern, mobile-first web application for tracking pet vaccinations and discovering veterinary services in Sri Lanka.

## Features
- **Pet Dashboard**: Manage profile and vaccination history.
- **Admin Panel**: Manage veterinary surgeons and platform statistics.
- **Vet Search**: Discover verified clinics nearby.
- **Responsive UI**: Optimized for all devices with a clean blue/white theme.

## Getting Started
Simply open `index.html` in any modern web browser to run the application locally.

## Version Control
To save your changes to GitHub easily:
1. Make your edits to the files.
2. Run the following command in your terminal:
   ```bash
   npm run save
   ```
   This will automatically stage all changes, commit them, and push them to your repository on the `main` branch.

## Deployment
This project is configured for **Render Blueprint** deployment.
- Connected via GitHub: Automatically deploys on every push to `main`.
- Site Config: See `render.yaml`.

## Project Structure
- `index.html`: Entry point.
- `style.css`: Design system and layout.
- `js/`: Application logic and page templates.
- `package.json`: Project metadata and scripts.
